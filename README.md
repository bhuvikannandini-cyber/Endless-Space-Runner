A 2D endless runner space game built using Python and Pygame.

About the Game
Endless Space Runner is a fast-paced arcade game where the player controls a space explorer flying through space while avoiding asteroids, aliens, black holes, and debris. The player collects stars, planets, and treasure to increase their score while the game speed and difficulty increase over time.

Features
- Procedurally generated obstacles and rewards
- Increasing difficulty over time
- Player movement & collision detection
- Scoring and progression system
- Boss fights (Version 2)
- Power-ups (shield, magnet, speed boost)
- Achievement system (8 achievements)
- Save system

Technologies Used
- Python  
- Pygame  
- Object-Oriented Programming
